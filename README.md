# Weather-Based Comms 

This is a web application that lets you know what the best communication channel is based on the weather. Everybody knows that people love being contacted in different ways depending on what Mother Nature throws their way!

**NOTE:** Currently only works if your contacting poeple in Minneapolis, MN.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
